About This Project

This website was created to provide quick access to important emergency numbers for countries like India and Canada. During emergencies, every second matters. This platform helps users quickly call police, ambulance, fire services, and other essential helplines.

Features include the Panic Button for instant WhatsApp SOS live location sharing, dynamic search, and a modern Dark/Light mode theme. The goal is to make everyone safer with technology.
